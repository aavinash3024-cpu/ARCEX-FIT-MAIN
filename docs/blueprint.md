# **App Name**: PulseFlow

## Core Features:

- Dashboard Overview Display: Visual presentation of key health metrics like Calories, Macros, Hydration, Tasks, Goals, Steps, and Weight in a dynamically scrollable and swipeable card interface.
- Nutrition Tracking Interface: User-friendly interface to log daily meals, capture details about meal components, and view 'Today's Meal', 'Calorie History', and a summary of 'Today's Analysis' through dedicated cards.
- Workout Management Display: Intuitive display of 'Workout Logging', 'Personal Records', an 'Exercise Library', and a detailed 'Workout History' within a structured layout.
- Split Program Builder: An interface for creating and organizing custom workout 'splits' or routines, with an initial focus on displaying the structure.
- Progress and Ranking Visualization: Visual cards dedicated to 'Weight Card' data and a dynamic 'Rank' section, allowing for a sliding page interaction between them for comparative views.
- Intelligent Goal & Task Tool: An AI-powered tool that provides personalized suggestions for daily tasks, hydration targets, and potential macro adjustments based on mock user health and activity data.

## Style Guidelines:

- Primary color: A muted blue-grey (#6A89A7) for a sophisticated and calm foundation.
- Secondary color: A very light, soft blue (#BDDDFC) to provide a clean and expansive background.
- Accent color: A brighter, sky-like blue (#88BDF2) for highlights and interactive elements.
- Dark accent/text color: A deep, rich blue-grey (#384959) for strong contrasts and readability.
- Headline and body font: 'Inter' (sans-serif) for its modern, clear, and objective aesthetic, ensuring optimal readability for data-rich content across all card types.
- Utilize icons with different gradients within a circular ring, reflecting the app's blue-grey color theme, to represent various metrics, tasks, and actions, ensuring visual clarity and modernity across all cards and sections.
- A card-based layout featuring well-defined sections (Dashboard, Nutrition, Workout, Rank), emphasizing generous spacing and visual hierarchy to enhance clarity. Cards for Calories/Macros should be presented in a horizontal, swipeable belt for ease of access.
- Implement subtle and fluid animations for card transitions, particularly for the dashboard's swipeable elements and the sliding page between Weight and Rank, to create a seamless and responsive user experience.
